# RFC 2131 - 动态主机配置协议（DHCP）

## 文档概述

RFC 2131是由Bucknell University的R. Droms于1997年3月发布的DHCP协议标准规范文档，标题为"Dynamic Host Configuration Protocol"。

该文档定义了动态主机配置协议（DHCP），为TCP/IP网络上的主机提供配置信息的框架。DHCP基于引导协议（BOOTP），增加了可重用网络地址的自动分配能力和额外的配置选项。

RFC 2131废弃了RFC 1541，是DHCP协议的标准规范。
